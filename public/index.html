X Live Chat Setup

X has no API for live-broadcast chat, so a small helper on the streaming PC relays it into the feed. Start it once and leave it — it attaches to each broadcast automatically.

  1. On the streaming PC, in the project folder, run npm run x-login once and sign into any X account (a burner works — it only reads public chat; the hosts never log in).
  2. Copy each host's command below, run it in a terminal, and leave it running during the show.

Add a watcher with an X handle to generate its command.
